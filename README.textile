A collection of projects in python looking at semantics. Mostly orientated around search.

* Latent semantic analysis - "http://www.joesniff.co.uk/projects/latent-semantic-analysis-in-python.html":http://www.joesniff.co.uk/projects/latent-semantic-analysis-in-python.html
